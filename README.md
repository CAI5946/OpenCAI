# Coding Agent Learn

这个目录用于系统学习前沿 Coding Agent 的工作原理，并逐步实现一个适配个人工作流的最小 Coding Agent。

## 当前锚点

- 主要研究对象：前沿 Coding Agent CLI (以 ReAct 模式为主)
- 研究方式：黑盒观察、公开文档、真实任务实验、行为对照
- 实现目标：借鉴 Claude Code 的核心开发工作流和可观察行为，产出个人可用的产品化 CLI 版最小 Coding Agent
- 当前路线：Phase 6 toy project closed loop 已完成，后续按 OpenCAI 主实现 + Claude Code reference pass 双轨推进 Phase 7-11

## 边界

- 不追求一次性做完整 Agent
- 不加入无明确用途的目录、框架或抽象
- 不复制 `claude-code/` 的代码或闭源实现，只学习架构、边界和行为原则
