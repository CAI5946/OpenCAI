# Coding Agent Learn

这个目录用于系统学习前沿 Coding Agent 的工作原理，并逐步实现一个适配个人工作流的最小 Coding Agent。

## 当前锚点

- 主要研究对象：前沿 Coding Agent CLI (以 ReAct 模式为主)
- 研究方式：黑盒观察、公开文档、真实任务实验、行为对照
- 实现目标：个人可用的最小 Coding Agent，而不是复刻特定商业产品

## 边界

- 不追求一次性做完整 Agent
- 不加入无明确用途的目录、框架或抽象
